
Geliştirdiğim yemek sipariş uygulaması, Clean Architecture ve MVVM mimarisi ile tasarlanmış, kullanıcı dostu bir deneyim sunuyor. Rest API ve Retrofit entegrasyonu sayesinde dinamik veri akışı sağlanırken, kullanıcılar yemekleri listeleyip arama yapabilir, fiyatlarına göre filtreleme ve sıralama yapabilirler. Ürünleri sepete ekleme, silme ve favorilere ekleme gibi özelliklerle deneyimlerini kişiselleştirme imkanı buluyorlar. Dependency Injection ile bağımlılıklar yönetilirken, Coroutines ile asenkron işlemler hızlandırılmıştır.
